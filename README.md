# 🍽️ Restaurant Data Analysis using Power BI

## 📌 Project Overview
This project analyzes restaurant data to generate insights using Power BI dashboards including sentiment analysis, product insights, customer segmentation, and strategy analysis.

---

## 🎯 Project Objective
- Analyze restaurant data for actionable insights  
- Perform sentiment analysis (Positive, Neutral, Negative)  
- Identify trends and top cuisines  
- Segment customers based on behavior  
- Support business decision making  

---

## 📊 Dataset
- Restaurant Name  
- Cuisine  
- Rating  
- Number of Ratings  
- Average Price  
- Area  
- Location  
- Pure Veg  
- Sentiment (Derived)  

---

## 🧹 Data Preprocessing
- Handled missing values  
- Removed duplicates  
- Created sentiment column based on rating
